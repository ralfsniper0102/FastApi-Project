# FastApi-Project 

Projeto backend utilizado para Cadastro de Usuários e Gerenciamento de Acesso.

## Funcionalidades
- Cadastro e consulta de usuários
- Autenticação JWT
- Persistencia dos dados no Banco de dados
- Documentado com Swagger
